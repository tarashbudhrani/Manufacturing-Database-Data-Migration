# Manufacturing Database & Data Migration

## Overview
This project focuses on designing and migrating relational database tables to support structured reporting and analytics. It includes schema mapping, data validation checks, and deployment-ready testing to ensure a smooth migration process.

## Key Features
- Designed and migrated relational database tables with clean schema mapping for reliable analytics and reporting.
- Validated data integrity using SQL checks (duplicates, null handling, referential integrity) to ensure accurate migration results.
- Documented the migration workflow and supported smooth deployment through structured testing and verification.

- SQL
- Relational Database Design
- Data Migration & Validation
